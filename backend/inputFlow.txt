2- Add-Backs Section (Dynamic / Repeatable Group)

This is where your current system breaks. You need a repeatable structured component, not free-form prompts.

Has Add-Backs?
Type: radio button (Yes / No)

If Yes, show a repeater list:

Each Add-Back Entry:

Description
Type: text input (string)
Validation: required
Amount
Type: currency input
Validation: numeric only (this prevents your exception)
Category
Type: select dropdown
Options:
discretionary
non-recurring
non-operational
accounting adjustment
(enum on backend)
Confidence Level
Type: slider (0–100)
Alternative: number input with step=1
Display: percentage
Add Another Add-Back
Type: button (adds new entry)
